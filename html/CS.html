Boolean Algebra

Truth Tables

A truth table consists of true/false values for Boolean variables. One row per each possible combination of arguments. Each row gives the unique function value for specified arguements. For N inputs there are 2^N rows.
The number of boolean functions in N-variables is 2^(2^N).

Sum of Products and Product of Sums

The Sum of Products expression is a Boolean expression where all the variables, both true and complemented are ANDed to form products and all the products are ORed together to form the expression. Each product term must contain all the variables, either in their true or complementary form. Any Boolean function can be described using a SOP expression.
The product term in a SOP expression is called an implicant, because it implies the expression is true when the product term is true. Each product term with all the variables is also called a minterm.
A literal is a variable that is in its true form or its complementary form.
Product of sums (POS) is a Boolean expression composed of ANDed (product) a collection of OR terms (sum). The duality of Boolean algebra is shown in the comparison of SOP and POS expressions.
In POS, each sum term identifies a zero of the truth table.
A maxterm contains all the variables in true or complementary form all ORed together. Each maxterm corresponds to a row in the truth table.

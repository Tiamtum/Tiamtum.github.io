Cool Websites

A list of interesting/useful websites that I like (in no particular order).

peelopaalu.neocities.org
-Link directory of old fashioned webpages and more.
gifcities.org
- GIF search engine.
learnxinyminutes.com
- Crash courses of many different computer languages and more.
opengameart.org
- Mostly free game development assets.
fonts2u.com
- Free fonts.
cs.usfca.edu/~galles/visualization/about.html
- Contains visualizations for many algorithms and data structures.
templeos.org
- The Third Temple.
winprog.org
- Tutorial series for the Win32 API.
tusaalanga.ca
- Resource for learning Inuktut.
codecogs.com/latex/eqneditor.php
- Utility for converting LaTeX into image files.
